## 模块 01：选题策划与爆款标题

### 高潜力选题方向

| 类型 | 示例 | 特点 |
|------|------|------|
| 痛点解决类 | "早八人5分钟快速出门妆" | 精准击中用户困扰 |
| 热点关联类 | "暖冬好物｜冬季办公室保暖神器" | 借助热点流量 |
| 经验分享类 | "做小红书3个月涨粉2w的技巧" | 真实感强 |
| 盘点清单类 | "10款学生党平价彩妆推荐" | 直观易懂 |
| 对比反差类 | "平价vs大牌彩妆持妆对比" | 引发好奇心 |

### 爆款标题逻辑

```
1. 数字公式：人群 + 数字 + 解决方案
   例："34岁职场人坚持6个习惯，轻松提升工作效率"

2. 疑问式：引发用户好奇心
   例："为什么你的小红书笔记没人看？"

3. 痛点式：直击核心需求
   例："熬夜党必看！拯救暗沉肌的3个小方法"

4. 对比式：制造反差感
   例："从0粉到10W粉，我只用了3个月"

5. 场景式：结合具体场景
   例："早八人快速出门妆，5分钟搞定"
```

## 模块 02：图文大纲与结构设计

### 爆文大纲模板（灵感倒推结构）

```
1. 3秒钩子开头 → 抓住注意力
2. 15秒场景代入 → 引发共鸣
3. 30秒解决方案 → 分点呈现
4. 互动引导结尾 → 提升互动率
```

### 4种结构模板

| 模板 | 适用场景 |
|------|----------|
| 清单体干货结构 | 好物推荐、清单盘点 |
| 场景化测评结构 | 产品测评、教程分享 |
| 经验分享型结构 | 成长记录、干货输出 |
| 视觉冲击型结构 | 改造前后、对比展示 |

## 模块 03：正文内容与图文制作

### 正文创作方法

```
1. 对话感文案：口语化、亲切
   例："宝子们，今天分享一个超级实用的小技巧"

2. 实战价值内容：真实有效
   例："这个技巧我亲测有效，曝光量提升了50%"

3. 短句分段：易于阅读
   例：每个要点单独一行，避免长段落

4. emoji增强表现力
   例："✨超级实用技巧🔥"、"💡亲测有效"
```

### 图文制作技巧

| 项目 | 规范 |
|------|------|
| 封面 | 3:4竖版，主体占60%，文字居中 |
| 配图 | 每点配一张相关图片，清晰美观 |
| AI辅助 | DALL·E 3生成插画 |
| 合规 | 火兔工具箱违禁词检测 |

## 模块 04：自动发布与SEO优化

### SEO优化方案

```
1. 关键词：标题+正文+标签
2. 标签：5-10个精准标签
3. 字数：300-800字
```

### 自动发布工具

- 火兔工具箱
- 薯秘书
- 设置发布间隔避免风控

### 运营优化

```
1. 数据分析：每周查看曝光高的选题
2. 爆款库：记录效果好的内容模式
3. 用户互动：根据评论迭代内容
```
